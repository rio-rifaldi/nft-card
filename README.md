## Run Locally

Clone the project

```bash
  git clone https://github.com/rio-rifaldi/nft-card.git
```

Go to the project directory

```bash
  cd nft-card
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run dev
```
